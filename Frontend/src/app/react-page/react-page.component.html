Perspective I Gained

Tools are just Tools

After following along a tutorial for Nextjs, Tailwind, and TypeScript I got perspective.

I am agnostic about frontend frameworks. Angular compared to React had opinionated difference on file structure and how to go about building a project. Overall, I am a better developer going through that, to reason and process how to build a product.

Also, I found alot of value from TypeScript and Tailwind CSS because they are much polished compared to their predecessors. Type safety and having a library full of utility of common use behavior made project development easier, cleaner, and easier to trouble shoot issues.

In the end, any framework can get a job done. Although there are considerations from a architectural and business standpoint. I can say for certain, that that the upsides for TypeScript far outweigh using JavaScript. I do not want to reinvent the wheel but I also rather use a tool that is objectively more useful. TypeScript is to JavaScript as C++ is to C. The difference is an upgrade because of how powerful the technology is evolving to solve problems in a Object Oriented Programming way.
